## Risks & Mitigations (Single-Module MVP)

| Risk | Impact | Mitigation | Owner | Status |
|------|--------|------------|-------|--------|
| Excel import complexity | High | Support CSV/JSON intermediate; Preview & Map UI for column mapping; Tenant-specific saved mappings (Option A) | Primary Agent | Not Occurred |
| Evidence storage costs | Medium | Implement retention policies early; Tiered S3 storage (hot/cold); Compress evidence files; 25MB limit for photos | DevOps Agent | Not Occurred |
| Mobile UX challenges | Medium | Early mobile testing; PWA optimization; Responsive design; Skeuomorphic feel per DR Hub PDF | UX Agent | Not Occurred |
| Email deliverability | Medium | Use established provider (SendGrid/SES); Monitor delivery rates; SPF/DKIM configured; Two-tier tracking loop | Integration Agent | Not Occurred |
| Pilot tenant delays | High | Identify 3 backup pilot tenants (High-Reg, Global Ops, Tech-Forward); Offer "Compliance Partnership" incentives | Primary Agent | Not Occurred |
| WebSocket scaling issues | High | Use Redis Pub/Sub for distributed WebSocket; Load balancer with sticky sessions; Tenant-isolated channels | Primary Agent | Not Occurred |
| Resource check-in adoption | Medium | SMS/WhatsApp primary channel (works when data networks fail); Reminder notifications; Escalation logic | UX Agent | Not Occurred |
| Circular dependency in DAG | Critical | Recursive check during task save; Visual error map for invalid dependencies; Prevent save until resolved | Primary Agent | Not Occurred |
| PDF report generation performance | Medium | Use headless browser (Puppeteer) with caching; Pre-generate report templates | UX Agent | Not Occurred |
| Feature flag misconfiguration | High | Admin UI with audit logging; Default flags set to false for MVP; Validation middleware | DevOps Agent | Not Occurred |

---

## Architectural Decisions

### Design Language: Skeuomorphism (Neumorphism)
- Chose a neumorphic UI style (soft shadows, extruded/inset surfaces) to provide a tactile, professional feel suitable for a resilience platform. This differentiates ResiliPath from standard flat-design SaaS competitors.

### Static Generation (SSG)
- Leveraging Next.js static generation for the landing page to ensure sub-100ms load times and SEO optimization, adhering to the "Free-Tier Priority" mandate by reducing server-side execution.

### PWA Strategy
- Implemented PWA features from Day 1 to support "Resilience" use cases where network connectivity might be degraded during an actual disaster recovery scenario.
